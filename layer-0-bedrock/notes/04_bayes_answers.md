# 0.4 Probability — 10 Bayes Answers

1) Bayes rule:
P(A|B) = P(B|A) * P(A) / P(B)

2) Meaning:
Update a probability after seeing new evidence.

3) Prior:
What you believed before new evidence.

4) Likelihood:
How likely the evidence is if the hypothesis is true.

5) Posterior:
What you believe after using the evidence.

6) Base rate:
How common something is before new evidence.

7) Why Bayes matters:
It stops you from ignoring the base rate.

8) Medical test example:
Even if a test is good, a rare disease can still mean many positives are false positives.

9) If P(B) is small:
Evidence B is rare, so it changes the result more.

10) Main lesson:
Do not confuse P(A|B) with P(B|A).
